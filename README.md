## Ktlint formatting 자동화

1. IntelliJ IDEA 상단 메뉴에서 [IntelliJ IDEA] - [Settings] 클릭
2. 왼쪽 패널에서 [Plugins] 클릭
3. [Marketplace] 탭에서 ktlint 검색 및 설치
4. [IntelliJ IDEA] - [Settings] - [Tools] - [ktlint] 클릭
5. 아래 사진처럼 설정하고 IntelliJ 다시 시작
